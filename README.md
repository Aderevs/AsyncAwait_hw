# AsyncAwait_hw
Завдання 2

Створіть WPF програму, розмістіть у вікні TextBox і дві кнопки. 
При натисканні на першу кнопку в TextBox виводиться повідомлення «Підключено до бази даних», 
при цьому в обробнику встановіть затримку в 3-5 сек для імітації підключення до БД, 
також ця кнопка запускає таймер, який з періодичністю в кілька секунд виводить у TextBox 
повідомлення «Дані отримані». При натисканні на другу кнопку за аналогією з першою відключаємось 
від бази (із затримкою), виводимо повідомлення та зупиняємо таймер.
